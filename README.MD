# WMS Practice API

REST API для системы управления складом (Warehouse Management System), разработанное в рамках учебной практики.

## 📋 Описание

Проект представляет собой бэкенд-сервис для учета товаров, движений на складе, инвентаризации и управления периодами. API предоставляет возможности для создания, чтения, обновления и удаления данных о складских операциях.

Интерфейс для тестирования API автоматически генерируется через **Swagger UI**.

## 🛠 Стек технологий

* **Язык:** Python 3.x
* **Фреймворк:** FastAPI
* **Документация:** OpenAPI 3.1 (Swagger)
* **База данных:** SQL Server Managemnent Studio 2021

## 📦 Эндпоинты (API)

Основные разделы API:

* **Products (Товары):** Управление списком товаров.
* **Movements (Движения):** Учет прихода и расхода.
* **Reports (Отчеты):** Получение данных по остаткам.
* **Periods (Периоды):** Управление отчетными периодами.
* **Auth (Авторизация):** Вход в систему.
* **Reservations (Резервирование):** Бронирование товаров.
* **Inventory (Инвентаризация):** Отправка данных инвентаризации.

## 🚀 Установка и запуск

### 1. Клонирование репозитория

```bash
git clone <url-твоего-репозитория>
cd WMS-Practice
```

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

* **Если файла requirements.txt нет, установите необходимые пакеты вручную, например: pip install fastapi uvicorn pydantic**

### 3. Запуск сервера

```bash
uvicorn main:app --reload
```

## Документация (Swagger)

### После запуска сервера документация API доступна по следующим ссылкам

1. Swagger UI: <http://localhost:8000/docs>
2. ReDoc: <http://localhost:8000/redoc>
3. OpenAPI JSON: <http://localhost:8000/openapi.json>
